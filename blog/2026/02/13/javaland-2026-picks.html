Ed's JavaLand 2026 Session Picks

Feb 13, 2026

Cross posted at dev.to.

I can hardly believe I’m writing my session picks for the 12th JavaLand conference!

In fact, I didn’t write it. I used Claude Cowork to write it and I am just reviewing the work and updating the style to be more Ed-like. This is the future of work! See below for my prompts and stuff. I’ll low-key indicate the parts I manually edited with an “eb:” prefix.

Having attended every single one since 2014, this conference holds a special place in my heart. From the early days at Phantasialand to last year’s successful second showing at Nürburgring, JavaLand continues to evolve while maintaining its unique charm and technical depth.

eb: As a practicing schlager fan, I am delighted to be visiting Europapark, shooting location for Immer wieder Sonntags. Perhaps the organizers will do something Schlager themed?

Tuesday, 10 March 2026

View Tuesday’s full program

08:30 - 09:10 Sanssouci - Generics – You Never Know What You’re Gonna Get

Program link

Bram Janssens tackles one of Java’s most misunderstood features with this Forrest Gump-inspired title. Generics have been with us since Java 5, but I still regularly see code that doesn’t leverage them properly. This session promises to clear up common misconceptions. eb: I hope he covers the mechanics of type erasure.

10:00 - 10:50 Dome - (KEYNOTE) AI: It AI-n’t What You Think!

Program link

This keynote from the legendary Dr. Venkat Subramaniam is going to be packed. I’ve had the pleasure of seeing Venkat speak many times over the years, and he never disappoints. His ability to cut through hype and deliver practical, deeply technical content is unmatched. Given the current AI frenzy, I expect Venkat will provide much-needed clarity and pragmatism. eb: I’m honored to even be on the same program as this legendary gem of our field.

11:00 - 11:40 Das Studio - ADR, MADR, eADR – Architekturentscheidungen sichtbar und nachvollziehbar gestalten

Program link

Architecture Decision Records are one of those fundamental practices that every serious architect should master. Dr. Oliver Kopp covers not just ADRs but also their variants MADR and eADR. Having worked on large-scale enterprise systems for decades, I can attest that documenting architectural decisions pays dividends years later when you’re trying to understand why a particular choice was made.

12:00 - 12:40 Magic Cinema - The Trust Trap - Security von Coding Assistants

Program link

eb: Clemends Hübner covers an absolutely vital topic. As economic forces make it less and less viable to avoid total dependence on AI for turning the crank on the SDLC, code security may be the last bastion of human indespensibility. Let’s see what Clemens has to say.

Tie: 13:00 - 13:40 Dome - The Past, the Present, and the Future of Enterprise Java

Program link

My colleague Ivar Grimstad delivers what promises to be an authoritative look at where Enterprise Java has been and where it’s going. As someone who has been working with Enterprise Java since the J2EE 1.2 days, I’m very interested in Ivar’s perspective on our collective journey and future direction. eb: fun fact, the first time I gave a talk with the “Past, Present, and the Future of [something]” was W-JAX 2005. Of course, I wasn’t the first to originate that title pattern.

Tie: 13:00 - 13:40 Sanssouci - Modern Java - Ask Me Anything

Program link

eb: This is a tough time slot. If you want server side, go to Ivar’s talk. If you want core Java, come here. Nicolai Parlog is an excellent presenter, and for many people is the face of Java on YouTube. Combine the Java savvy of the JavaLand audience with Nicolai on stage in AMA format, and you’ll surely learn a lot.

14:00 - 14:40 Dome - Von Spaghetti Code zu Halluzinations-Lasagne: Baupläne für Software-Architektur 3.0

Program link

Ingo Eichhorst’s provocatively titled talk about modern software architecture in the age of AI looks fascinating. The metaphor alone is worth attending for, but I expect the content will deliver substantive insights into how AI is changing our approach to architecture. eb: I’m very curious to see how well what the author says about the extent of hallucinations holds up. As was documented in this famous (and famous for being famous) essay, and as confirmed in my own lived experience, hallucinations are much less of a problem in models since Claude Sonnet 4.5.

15:00 - 15:40 Sanssouci - Reset fürs Lernen – Warum KI uns zwingt, Weiterbildung neu zu erfinden

Program link

eb: To survive the AI transition, developers need to once again reshape how they learn. I’m old enough to remember the technical book section in a physical book store as the way we learn. Then it became tutorials. And now it’s different again. This session from Michaela Schrötel seems essential to me.

16:00 - 16:40 Magic Cinema - 82 Bugs I Collected in a Year You Won’t Believe Made It to Production

Program link

François Martin’s session looks like it will be both educational and entertaining. Learning from production bugs is one of the most effective ways to improve your craft. eb: I’m a big fan of these “here’s a bunch of bugs” sessions.

17:00 - 17:40 Bellevue - Methodology Panel:The Mindset of Software Development: Practical Strategies for Developers

Program link

eb: Panel sessions always depend on the quality of the panelists, and the panelists for this session are not listed. However, I know Ixchel and she has a great network and attracts talent. With her at the helm, this is sure to be compelling.

Wednesday, 11 March 2026

View Wednesday’s full program

Tie: 09:00 - 09:40 Bellevue - Jakarta Data - Das Ende des Boilerplate Code?

Program link

Thorben Janssen is an expert on Jakarta EE and persistence technologies. Jakarta Data promises to reduce boilerplate code in data access layers, and I’m eager to see how this specification will help developers. eb: As the co-release coordinator for Jakarta EE 11, which includes Jakarta Data, I can’t wait to find out how popular it is becomming.

Tie: 09:00 - 09:40 Dome - Observing Project Valhalla

Program link

My old friend Cay Horstmann takes on Project Valhalla, one of the most ambitious projects in Java’s history. Cay has been a fixture in the Java community for decades, and his educational expertise makes him the perfect person to explain these complex concepts. Valhalla’s value types and primitive classes will fundamentally change how we write high-performance Java code. eb: Cay is an experienced teacher and lecturer, you’ll learn a lot.

10:00 - 10:40 CircusCelebration - Legacy Architecture Refactoring mit ArchUnit

Program link

Daniel Rosowski demonstrates using ArchUnit for legacy architecture refactoring. ArchUnit is a powerful tool for enforcing architectural constraints through tests, and applying it to legacy systems is particularly valuable.

12:00 - 12:40 Versailles - Secrets of Performance Tuning Java on Kubernetes: Leyden, Heap Sizing and Beyond

Program link

My colleague Bruno Borges covers performance tuning for Java on Kubernetes, including Project Leyden. Bruno and I worked together back when he was with Oracle, and I’ve always appreciated his deep technical knowledge combined with practical applicability.

14:00 - 14:40 Dome - The Evolution of Design Patterns in the Age of AI

Program link

eb: I fondly remember the days when the pace of change of the practice of development was such that there was time for experts to come together and write books about the practice of development. Books such as “Design Patterns”. Those days appear to be gone, and are likely not to return. But that doesn’t stop Miro! Miro is undaunted and brings the pattern of doing a talk about design patterns to the domain of contemporary software development. I’m very curious.

15:00 - 15:40 Das Studio - AWS vs Azure: The Key Differences That Matter

Program link

eb: I must tread carefully with this session. Because I’ve been working for the last six years, and given many talks, here and elsewhere, about my work to make Azure the best cloud for Java developers, I’m very curious to see what Artun Subasi has to say. From the abstract, it seems language agnostic.

16:00 - 16:40 Dome - From CRUD to Cognitive: Modernizing Legacy Java Apps with Embedded AI

Program link

This is my own session where I explore how to modernize traditional CRUD applications by embedding AI capabilities. I’ll show practical approaches to taking existing Java EE/Jakarta EE applications and adding intelligent features without complete rewrites.

eb: If I wasn’t presenting at this time, I’d definitely attend Hilfe, ich habe den KI-Hype verpasst! Alles was man wissen muss in 105 Minuten from Florian Sommer.

17:00 - 17:40 CircusCelebration - Deterministische Unit Tests für Multi-Threaded Java

Program link

Thomas Krieger tackles the challenging problem of writing deterministic unit tests for multi-threaded code. With virtual threads making concurrent programming more accessible, this skill is increasingly important.

17:00 - 17:40 Magic Cinema - API = Some REST and HTTP, right? RIGHT?!

Program link

Rustam Mehmandarov challenges our assumptions about APIs. Given how many developers conflate REST with HTTP and miss the broader API landscape (GraphQL, gRPC, etc.), this session should provide valuable perspective. eb: good to see HATEOAS (which I like pronouncing HATE ee Ohs to rhyme with Cheerios) getting some attention in 2026.

See you in Rust in March!
